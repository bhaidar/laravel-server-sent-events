In this application, I demonstrate using server-sent events in a Laravel application. 

I've chosen to use OpenAI API to stream a chat via the server-sent events feature.

The application represents a simple chat page that you ask questions and OpenAI responds with an answer. The answers are stream as they are received from the API, using server-sent events.


** Disclaimer**
This application is just a demo and not suitable for production.
